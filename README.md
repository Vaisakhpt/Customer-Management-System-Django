\# Customer Management System - Django



A Django-based CRM application for managing customers, products, and orders.



\## Features

\- Customer, Product, Order management

\- Django ORM database models

\- ForeignKey and ManyToMany relationships

\- CRUD operations using Django ModelForms

\- Order filtering functionality

\- SQLite database integration



\## Tech Stack

\- Python

\- Django

\- SQLite

\- HTML

\- CSS
